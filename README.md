# SmartInfra Security Suite

A workflow-orchestrated cybersecurity assessment system for automated network risk analysis.

## Overview
This repository provides supporting materials for the SmartInfra Security Suite, including workflow configurations and sample outputs used in the study.

## Contents

### workflows/
Contains n8n workflow configurations for each analytical level:
- L0: Asset Discovery
- L1: Port Discovery
- L2: Service Detection
- L3: Risk Detection and CVE Intelligence
- L4: Report and Notification
- L5: Security Intelligence

### reports/
Contains sample security assessment reports generated by the system.

## Reproducibility
The provided workflows can be imported into n8n to reproduce the analysis pipeline described in the paper.

## Reference
